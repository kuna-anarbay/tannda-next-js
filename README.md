#### This is a [tannda.kz](https://tannda.kz/) platform 

## Getting Started

First, run the development server:

```bash
npm install 
# and
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Frameworks and libraries

- [Redux-requests](https://redux-requests.klisiczynski.com/): Used for networking in redux
- [Sass](https://sass-lang.com): For styling
- [Next.js](https://nextjs.org): For routing, heads and pages
- [Typescript](https://www.typescriptlang.org): Language
- [React-icons](https://react-icons.github.io/react-icons): Icons lib
